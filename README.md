# Wormix‑Like – 2D Turn‑Based Artillery Game

## Overview

Original 2D turn‑based artillery game built with Phaser (web‑first) and FastAPI (backend).  
Content‑first architecture with structured data files.  
Electron support is kept optional for future desktop packaging.

## Prerequisites

- Node.js **20+** (LTS recommended)
- Python **3.11+** (3.10 should also work)
- npm (comes with Node.js)
